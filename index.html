Utilized FAISS vector storage and Sentence Transformers to efficiently retrieve 500+ document chunks, enabling query processing in under 2 seconds. Developed an interactive UI using Gradio and integrated LangChain for seamless PDF document processing, achieving 85%+ response relevance.

Integrated Vehicle Safety Alert System (IVSAS)

Developed an Integrated Vehicle Safety Alert System (IVSAS) to provide real-time alerts about accident-prone areas using Arduino UNO, ESP32, RF communication, and GPS modules. Implemented dual-notification alerts with an audio warning and mobile notifications via the Blynk app, enhancing driver awareness and road safety. Successfully tested with reliable RF signal transmission up to 15 meters and GPS accuracy within ±5 meters.

Processed 1,300+ insurance records and implemented feature engineering for improved model accuracy. Evaluated cost distribution trends to enhance decision-making in insurance pricing.
